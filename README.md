# 日群友
你在干什么，你在干神魔？  

多的不嗦 不是你死 就是我亡  
新的一天从随机向群友发情开始！
## 使用方法
部署到 /plugins/example

然后在你受不了的群里发送
```
日群友
```
或者你也可以选择被群友日
```
日我
```
